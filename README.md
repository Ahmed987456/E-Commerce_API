# E-Commerce API

## Technologies
- ASP.NET Core 9
- Entity Framework Core
- SQL Server
- JWT Authentication
- BCrypt Password Hashing
- AutoMapper
- Swagger

## Features
- JWT Authentication & Authorization
- Role-Based Access (Admin / Customer)
- Products & Categories with SubCategories
- Shopping Cart
- Orders Management
- Password Hashing

## How To Run
1. Clone the repo
2. Update connection string in appsettings.json
3. Run: Update-Database
4. Run the project
